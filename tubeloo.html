TubeLoo

The Coolest YouTube Downloader for Python!

Are you tired of watching videos online? Want to save them forever? TubeLoo is here to save the day!

Getting started

Clone the project

git clone https://github.com/iamvictormwangi/tubeloo

Make the tubeloo executable

cd tubeloo



chmod +x tubeloo

Move the tubeloo file to path

mv tubeloo /usr/local/bin

That's it, tubeloo is installed

Usage

tubeloo 'http://link-here'

Download a playlist

tubeloo --p 'http://link-here'
